# Create Your Bot Like Akaragi
<details>
  <summary>Steps To Create Bot</summary>

First: Get Your Api Key And Brain id at [brainshop.ai](https://brainshop.ai)
Second: Get Token From [BotPapa](https://t.me/botfather)
```
$ git clone https://github.com/nekozu/AkaragiChatBot
$ cd akaragichatbot
$ npm i
```
Replace your bot token. brain id. and brainshop apikey at .env file. Done? Them
```
$ node index.js
```
  
Done!
